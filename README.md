# MisterConscio dotfiles
## Instalation:

The repo it's supposed to be used with [GNU/stow](https://www.gnu.org/software/stow/).

```sh
git clone https://github.com/MisterConscio/dotfiles.git
cd dotfiles
stow i3
```
